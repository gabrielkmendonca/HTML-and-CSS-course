Main Text Formattings

Bold / Highlight

In this sentence we have a term in bold using the <b> tag (non-semantic tag)

In this sentence we have a highlighted term using the <strong> tag (semantic tag)

Italics / Emphasis

In this sentence we have a term in italics using the <i> tag (non-semantic tag)

In this sentence we have a term in emphasis using the <em> tag (semantic tag)

Marked Text

We can also create a marked text with the <mark> tag

Big and Small Text

We are creating a big text and a small text in this paragraph using <big> and <small> tags

Deleted Text

We can mark a text as deleted to indicate that it should be read but not considered using <del>.

Inserted Text

We can mark text as inserted to emphasize it and indicate that it was inserted later <ins> (semantic tag).

There is also a way to make text underlined with <u> tag (non-semantic tag)

Superscript Text

To insert things like X20+3 we use the <sup> tag

Underwritten Text

To insert things like H2O we use the <sub> tag
